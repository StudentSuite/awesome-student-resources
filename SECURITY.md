# Security Policy

## Scope

This repository is a curated list of links. It hosts no application code and
ships no runnable software of its own. The tools, channels, and books it points
to are owned, maintained, and secured by their respective providers.

## A problem with a listed entry

If a listed resource has been compromised, turned malicious, or its link now
points somewhere unsafe:

- Open a [broken or outdated entry issue](../../issues/new/choose), or
- Open a pull request removing the entry.

## A problem with this repository

For anything affecting this repository itself (the CI workflows, the GitHub
Actions configuration, or a compromised link in our own docs), contact the
[StudentSuite](https://github.com/StudentSuite) organization through GitHub, or
open a private security advisory on this repo. We will respond as promptly as we
can.

## A note for users

This list curates links; it does not vet or host the tools behind them. Treat
third-party sites with normal caution: check pricing before paying, and don't
enter credentials on a site you don't recognize. See the "A Note on Links"
section in the README.
